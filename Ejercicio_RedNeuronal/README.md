Se implemento el ejercicio de una puerta lógica AND usando una neurona artificial (Perceptrón) La cual funciona como la unidad básica de las redes neuronales y se utiliza para tareas de clasificación lineal.
En el ejercicio se realizo un Script para comparar el enfoque del perceptrón entrenado con la tabla de verdad AND.

Para el ejercicio se coloca inicialmente los pesos y el sesgo en 0, los cuales son las variables con la que la neurona combina una entrada, para predecir la respuesta de esta, es 1 o 0.
Colocamos en la variable de épocas, un valor de 20, las cuales son la cantidad de veces que la neurona va a recorrer los datos para entrenarse y cuando el error sea igual a 0, se detendrá para dar una respuesta clave.

Ejecución:
Al momento de ejecutar el algoritmo, podemos ver que una tabla con los siguientes datos: (x1 x2 | y_esp y_entrenado).

Tenemos a X1 y X2 como los valores de entrada. Y_esp son los valores de salida con los que nosotros entramos el modelo. Y_entrenado son los valores que el modelo predice en base al aprendizaje supervisado ya realizado.

Luego nos da la oportunidad de digitar dos valores separados por espacio, y el modelo nos dará el resultado esperado para esos dos valores.
